# stu-ssm

#### 介绍
学生信息管理系统，基于Vue3、Element Plus、Spring、SpringMVC、MyBatis等框架开发的小项目，分为前端项目和后端项目。具有用户登录和退出、学生信息的创建、修改、删除和查询以及学生数据统计的功能。

```

```
再用IDEA打开，执行如下命令：
```
npm install
```
